# Westerfield

## Development server

Run `nx serve cloud` for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

## Codespace port forwarding

Run `nx serve cloud --host 0.0.0.0` if Codespace doesn't automatically forward the application port.

## Understand this workspace

This workspace is managed with [NX Workspace](https://nx.app)

### Graphing

Run `nx graph` to see a diagram of the dependencies of the projects.

### Remote caching

Run `npx nx connect-to-nx-cloud` to enable [remote caching](https://nx.app) and make CI faster.

### Further help

Visit the [Nx Documentation](https://nx.dev) to learn more.
